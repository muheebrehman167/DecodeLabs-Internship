# Expense Tracker Pro — Advanced GUI Edition

**DecodeLabs | Batch 2026 | Project 2**

## Objective

A professional desktop Expense Tracker application built with Tkinter, featuring real-time expense management, dashboard statistics, category-based tracking, persistent JSON storage, and a modern dark-themed user interface.

---

## Features

* Add new expenses with category selection
* Edit existing expense records
* Delete selected expenses
* Clear all expenses with confirmation prompt
* Real-time dashboard displaying:
  * Total Expenses
  * Total Transactions
* Expense Summary window with statistics
* Automatic date and time tracking for every expense
* Persistent storage using JSON (`expenses.json`)
* Auto-save and auto-load functionality
* Modern dark-themed user interface
* User-friendly expense table with scrollbar support

---

## Technologies Used

* Python 3.x
* Tkinter (GUI — standard library)
* JSON (data persistence — standard library)
* Datetime (timestamps — standard library)
