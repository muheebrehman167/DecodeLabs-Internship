# To-Do List Manager — Advanced GUI Edition
### DecodeLabs | Batch 2026 | Project 1
 
## Objective
A professional desktop To-Do List application built with
Tkinter, featuring persistent storage, task editing, and a
modern dark-themed interface.
 
## Features
- Add, edit, delete, and complete tasks
- Persistent storage using JSON (tasks.json)
- Live status bar showing Total / Pending / Completed counts
- Modern dark UI with custom color palette
- Double-click to quickly toggle task completion
- 'Clear Completed' bulk action
- Built using Object-Oriented Programming (OOP)
 
## Technologies Used
- Python 3.x
- Tkinter (GUI — standard library)
- JSON (persistence — standard library)
- datetime (timestamps — standard library)
 
## Installation & Usage
```bash
git clone https://github.com/muheebrehman167/todo-list-python.git
cd todo-list-python
python todo_gui.py
```

- Object-Oriented Programming (classes, methods, self)
- GUI development with Tkinter
- Data persistence using JSON
- Event-driven programming and lambda functions
